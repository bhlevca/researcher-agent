⚙️ LLM Parameters

Temperature0.4
Controls randomness. 0 = deterministic, 1 = creative. Lower values for factual tasks, higher for brainstorming.
Top K40
Limits token selection to K most likely tokens. Lower = more focused, higher = more diverse. 0 = disabled.
Top P0.9
Nucleus sampling — considers tokens whose cumulative probability reaches P. Lower = more focused. Works with Top K.
Min P0.0
Minimum token probability threshold relative to the top token. Filters out unlikely tokens. 0 = disabled.
Repeat Penalty1.1
Penalizes token repetition. 1.0 = no penalty, higher = less repetition. Useful for reducing loops.
Frequency Penalty0.0
Reduces repeated tokens proportional to frequency. 0 = disabled. Helps prevent word/phrase repetition.
Presence Penalty0.0
Encourages new topics. Penalizes any token that appeared regardless of count. 0 = disabled.
Max Tokens (num_predict)4096
Maximum number of tokens to generate per response. Higher = longer answers but slower. -1 = unlimited.
Context Length (num_ctx)32768
Context window size in tokens. Larger = more conversation memory but uses more VRAM.
Seed0
Random seed for reproducible outputs. 0 = random each time. Set a fixed number for consistent results.
Planning Max Attempts3
Maximum planning attempts before the agent proceeds. Higher = more retries if planning fails. 1-5 recommended.

Target Width2048

⚙️ Composer LLM Parameters

Temperature0.7
Controls randomness. 0 = deterministic, 1 = creative. Higher for more varied compositions.
Top K40
Limits token selection to K most likely tokens. Lower = more focused, higher = more diverse.
Top P0.9
Nucleus sampling — considers tokens whose cumulative probability reaches P.
Min P0.0
Minimum token probability threshold. Filters out unlikely tokens. 0 = disabled.
Repeat Penalty1.0
Penalizes token repetition. 1.0 = no penalty. Keep at 1.0 for Composer — JSON scores repeat "pitch", "duration" tokens naturally.
Frequency Penalty0.0
Reduces repeated tokens proportional to frequency. 0 = disabled.
Presence Penalty0.0
Encourages new topics. Penalizes any token that appeared. 0 = disabled.
Max Tokens (num_predict)16384
Maximum tokens per response. Composer needs 16384+ for thinking + JSON scores. Higher = longer scores but slower.
Context Length (num_ctx)32768
Context window size in tokens. Larger = more conversation memory but uses more VRAM.
Seed0
Random seed for reproducibility. 0 = random each time.
